# 🚀 DevHub: Full-Stack Events Platform

DevHub is a production-ready events management platform built with the latest **Next.js 16** features. This project demonstrates my ability to build scalable web applications using Server Components, real-time data fetching, and high-performance styling.

## ✨ Key Features
- **Dynamic Routing**: Utilizes Next.js App Router for optimized, file-based navigation.
- **Server-Side Rendering (SSR)**: Leveraging Server Components to reduce client-side JavaScript and improve SEO.
- **MongoDB Integration**: Real-time data storage and retrieval for event listings and details.
- **Partial Prerendering (PPR)**: A hybrid approach to rendering that blends static and dynamic content for a 10/10 user experience.
- **Responsive UI**: Styled with Tailwind CSS for a mobile-first, professional design.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (React 19)
- **Database**: MongoDB Atlas
- **Styling**: Tailwind CSS
- **Deployment**: Vercel (Edge Network Optimized)

## 🏗️ Getting Started
1. Clone the repo: `git clone https://github.com/Bethsol/dev-hub-nextjs16`
2. Install dependencies: `npm install`
3. Add your `MONGODB_URI` to a `.env` file.
4. Run development: `npm run dev`
